던전 크기 비교 — 점진적 증가 방식

실제 SVG 출력 크기로 그려놨어. 챗봇 채팅창에 들어갈 때 어떻게 보일지 가늠해봐.

방식 A: 점진적 증가 (10층마다 +1)

아인크라드 컨셉 — 위층 갈수록 넓어짐

1층

6×5 그리드 · SVG 320×280px

25층 (중간)

9×7 그리드 · SVG 470×400px

50층 (최종 보스 / 최대 크기)

11×10 그리드 · SVG 560×520px
⚠ 챗봇 환경 고려사항:
바베챗 같은 메신저는 이미지 높이 한계가 있어 (대략 750~850px 정도, 확인 불가).
• 1층 (320×280): 여유 ✅
• 25층 (470×400): 여유 ✅
• 50층 (560×520): 여유 ✅

즉 점진적 증가(11×10까지)는 채팅창 안전 범위 안이야. 만약 더 키우고 싶다면(15×12 같은) 그땐 한계 넘을 수 있어.

방식 B: 더 단순한 두 단계

A안이 복잡하면 이렇게 단순화도 가능

단순 2단계

• 1~25층: 7×6 그리드 (작음)
• 26~50층: 10×8 그리드 (큼)

→ 코드 단순, 변화는 한 번만

고정 크기

• 전 층 8×6 그리드 (중간)

→ 가장 단순, 시각적 통일감
→ 단점: 50층이라는 스케일감 없음
